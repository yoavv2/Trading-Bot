<!-- Analytics response consumed here (backtest block, AFTER 16-01 adds equity_curve). -->
GET /api/v1/analytics/strategies/{strategyId}?backtest_run_id={runId} → 
```typescript
type EquityPoint = {
  session_date: string;   // "YYYY-MM-DD"
  total_equity: number;   // chart Y axis
  cash: number; gross_exposure: number;
  realized_pnl: number; unrealized_pnl: number; open_positions: number;
};
type BacktestBlock = {
  run_id: string; status: string; started_at: string; completed_at: string | null;
  summary: Record<string, unknown>;   // includes starting_capital, ending_equity (raw, display as-is)
  metrics: Record<string, unknown>;   // see metric keys below
  equity_curve: EquityPoint[];        // added by 16-01; may be absent/empty for a given run
};
type AnalyticsResponse = { strategy: Record<string, unknown>; backtest: BacktestBlock | null; paper: unknown };
```

ANLX-02 metric key mapping (from backtest.metrics — source verbatim, do NOT recompute):
- Sharpe        → metrics.sharpe_ratio       (number)
- Max drawdown  → metrics.max_drawdown_pct   (number, label "Max drawdown %")
- Win rate      → metrics.win_rate_pct       (number, label "Win rate %")
- P&L           → metrics.total_return_pct   (number, label "Total return %") — the backend's own
                  return figure; there is NO dollar net-P&L field, and CONTEXT forbids recomputing one.
                  You MAY additionally show summary.starting_capital / summary.ending_equity verbatim
                  as raw context, but do NOT compute ending − starting.
- Trade count   → metrics.trade_count        (number)

Existing shared pieces to reuse (do not reimplement):
- useApiQuery<T>(endpoint) → { loading, result, refetch }; result is ApiResult<T> with
  result.ok, result.data, result.asOf, or failure fields.
- ErrorState({ failure, title? }) — the ONLY approved failure renderer; names the endpoint.
- FetchMeta({ asOf, loading, onRefresh }) — as-of timestamp + refresh control.
See console/src/components/runs/detail/MetricsPanel.tsx for the exact run-type-gated,
single-fetch usage pattern to mirror.
