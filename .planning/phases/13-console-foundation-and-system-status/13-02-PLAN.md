<objective>
Scaffold the read-only Next.js operator console at `console/` with an env-driven proxy to the FastAPI read surface and a documented single start command (CONS-01).

Purpose: Every later v1.2 screen lives inside this shell. The proxy decision matters: the FastAPI app has NO CORS middleware and adding one is not authorized (only the kill-switch route exception is). Routing all browser fetches through Next.js rewrites (`/backend/:path*` → API base URL) sidesteps CORS entirely while keeping the base URL in local env config.
Output: Running Next.js app shell (dark, minimal, debugging-instrument aesthetic) with nav, proxy wiring, `.env.example`, README instructions, and a `make console` target.
</objective>
